What's included

  • iris.app — System tray daemon with a GUI dashboard for managing context indexes, monitoring sessions, and configuring the prefetch engine.
  • iris CLI — Command-line interface for indexing, querying, and running iris as an MCP server inside Claude Code, Cursor, Copilot, and other AI coding tools.

After installation

  • Open iris from your Applications folder or Spotlight.
  • The system tray icon gives you quick access to daemon status.
  • In your terminal, run iris --help to get started with the CLI.
  • Run iris init in any project to set up MCP configs.

Requirements

macOS 13 Ventura or later. Apple Silicon or Intel.
