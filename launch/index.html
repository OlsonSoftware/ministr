iris

MCP server that manages LLM agent context windows like a CPU cache controller. Semantic search, code symbols, cross-language bridges, and budget management.

View on GitHub Download
cargo install --git https://github.com/alrik/iris-rs iris-cli
🔍

Semantic Search

Multi-resolution index with doc summaries, sections, claims, and embeddings. Hybrid BM25 + vector search with reranking.

🧬

Code Intelligence

Tree-sitter parsing for 12 languages. Symbol extraction, cross-references, and AST-aware chunking.

🌉

Cross-Language Bridges

Automatically maps function calls across language boundaries — Tauri commands, FFI, WASM bindings.

📊

Budget Management

Token-aware context window tracking with pressure levels, eviction candidates, and extractive compression.

🔮

Predictive Prefetch

Sequential, topical, and structural prefetch strategies warm the cache before the agent asks.

🔄

Live Coherence

File watcher detects changes and re-indexes in real time. Stale content alerts keep agents current.

Get Started in 30 Seconds

# .iris.toml in your project root
[[corpus]]
paths = ["src/", "docs/"]
ignore = ["target/", "node_modules/"]

[[corpus.git]]
repo = "https://github.com/example/shared-types"
paths = ["src/"]

# Add to your MCP client config (.mcp.json)
{
  "mcpServers": {
    "iris": {
      "command": "iris",
      "args": ["serve", "--transport", "stdio"]
    }
  }
}
