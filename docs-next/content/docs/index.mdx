ministr is an MCP server that gives AI agents real codebase intelligence: semantic search across code and docs, symbol-level navigation, cross-language bridge detection (Tauri, napi-rs, PyO3, wasm-bindgen, HTTP routes, FFI), and session memory so the same context isn't re-fetched on every turn.

## What ministr gives your agent

- **Semantic search** across your codebase and docs via `ministr_survey`
- **Code symbol navigation** — find and trace functions, types, callers via `ministr_symbols` / `ministr_references`
- **Session memory** — remembers what it sent; skips repeats; ships deltas on change
- **Predictive prefetch** — warms what it thinks you'll ask for next
- **Budget awareness** — tracks usage; compresses instead of silently dropping
