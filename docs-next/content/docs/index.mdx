ministr is a code intelligence MCP server. It gives AI coding agents AST-level understanding of your codebase: semantic search across code and docs, symbol-level navigation, real reference graphs, and cross-language bridge detection (Tauri, napi-rs, PyO3, wasm-bindgen, HTTP routes, FFI) — across ~29 languages, all local.

## What ministr gives your agent

- **Semantic search** across your codebase and docs via `ministr_survey`
- **Symbol navigation** — find and trace functions, types, and callers via `ministr_symbols` / `ministr_references`
- **Cross-language bridge detection** — links Tauri / napi-rs / PyO3 / wasm-bindgen / gRPC / HTTP / FFI boundaries via `ministr_bridge`
- **Multi-resolution results** — summary, section, claim, symbol, or full source — the slice that matters, not a file dump
- **Efficient delivery (a bonus)** — remembers what it already sent and ships only what changed on the next turn
