---
title: ministr_symbols
description: Search the code symbol index for structs, functions, traits, enums, and other declarations by name, kind, module, or visibility.
---

## Parameters

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `query` | string | no | — | Fuzzy name search (case-insensitive substring match) |
| `kind` | string | no | — | Filter by kind: `function`, `struct`, `trait`, `enum`, `impl`, `const`, `static`, `type`, `mod` |
| `module` | string | no | — | Module path prefix filter (e.g. `config` matches `config::sub`) |
| `visibility` | string | no | — | Filter by visibility: `pub`, `pub(crate)`, `pub(super)`, or `""` for private |
| `offset` | integer | no | 0 | Number of entries to skip |
| `limit` | integer | no | 100 | Maximum number of entries to return |

## Response

Wrapped in the [standard response envelope](/docs/tools#response-envelope). The `result` payload:

```json
{
  "budget_status": { /* see envelope */ },
  "result": {
    "symbols": [
      {
        "id": "sym-src/auth.rs::auth::validate_token",
        "name": "validate_token",
        "kind": "function",
        "file": "src/auth.rs",
        "line": 42,
        "signature": "pub fn validate_token(token: &str) -> Result<Claims>",
        "doc_preview": "Validates a JWT token against the configured public key.",
        "complexity": 4,
        "caller_count": 12
      }
    ],
    "total": 8,
    "offset": 0
  }
}
```

### `result` fields

| Field | Description |
|---|---|
| `symbols[].id` | Symbol ID — use with `ministr_definition` and `ministr_references` |
| `symbols[].name` | Declared name |
| `symbols[].kind` | Symbol kind (`function`, `struct`, `trait`, `enum`, `impl`, `const`, `static`, `type`, `mod`) |
| `symbols[].file` | Source file path |
| `symbols[].line` | 1-based start line in the source file |
| `symbols[].signature` | Declaration signature |
| `symbols[].doc_preview` | Optional — first line of the symbol's doc comment |
| `symbols[].complexity` | Optional — cyclomatic complexity (functions only) |
| `symbols[].caller_count` | Optional — transitive caller count for impact analysis |
| `total` | Total matches before pagination |
| `offset` | Offset of the first returned entry within the full result list |

## Behavior

- The query is tokenized with CamelCase + snake_case splitting (`ToolRouter` → `["tool", "router"]`, `list_symbols` → `["list", "symbols"]`); tokens shorter than two characters are dropped.
- All tokens must match (AND); if AND yields nothing on a multi-token query, ministr falls back to OR.
- Use `ministr_definition` to get the full source of a symbol.
- Use `ministr_references` to find callers, implementors, and importers.
