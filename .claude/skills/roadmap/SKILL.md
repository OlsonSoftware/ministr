Load the project's living design specification into context.

## Instructions

Read `.magistr/project.json` to get the spec file path (default: `ROADMAP.md`). Load and present the contents.

1. Read `.magistr/project.json` to get `spec.file`
2. Read the spec file from the project root
3. Present a brief summary: current phase, what's been completed, what's next
4. Answer any follow-up questions about the roadmap
