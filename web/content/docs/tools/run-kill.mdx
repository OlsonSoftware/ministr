Cancels a running run. On Unix the whole process group is killed, so child
processes spawned by the command are terminated as well.

## Returns

A `RunKillResponse` with:

- `run_id`
- `killed` — true when the run was still active and cancellation was requested

## Notes

- On Unix the whole process group is killed.
- On Windows only the direct child process is killed (a documented gap).
- Marked destructive.
