---
title: ministr_solid
description: Detect possible SOLID-principle violations deterministically.
---

Detect possible SOLID-principle violations across the codebase
deterministically. Returns clusters / findings labelled by principle
(`dry_ocp`, `srp`, `isp`, `dip`). Filter by kind/module and toggle principles
via params. Pair with `ministr_references` before refactoring.

Detection uses thresholded graph and cosine-similarity signals — no LLM
judgement.

## Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `kind` | string | — | Optional symbol kind filter. |
| `module` | string | — | Optional module filter. |
| `principles` | string[] | all six | Principles to run: `dry_ocp`, `srp`, `isp`, `dip`, `shotgun_surgery`, `cyclic_dependency`. Empty = all six. |
| `container_kinds` | string[] | `['impl','struct','class','mod']` | Symbol kinds treated as containers. |
| `interface_kinds` | string[] | `['trait','interface','protocol']` | Symbol kinds treated as interfaces. |
| `similarity_threshold` | number | `0.86` | Cosine-similarity threshold. |
| `jaccard_threshold` | number | `0.4` | Jaccard threshold. |
| `srp_cohesion_threshold` | number | `0.7` | Cohesion threshold for SRP findings. |
| `isp_min_methods` | number | `6` | Minimum methods for ISP findings. |
| `isp_max_overlap_fraction` | number | `0.33` | Maximum overlap fraction for ISP findings. |
| `min_lines` | number | `5` | Minimum symbol size in lines. |
| `limit` | number | `50` | Maximum findings, capped at 500. |
| `max_pairs` | number | `100000` | Maximum candidate pairs considered. |
| `representative_count` | number | `5` | Representatives reported per finding; overflow reported via `*_omitted`. |
| `shotgun_min_sites` | number | `3` | Minimum sites for shotgun-surgery findings. |
| `shotgun_max_jaccard` | number | `0.5` | Maximum Jaccard for shotgun-surgery findings. |
| `shotgun_min_packages` | number | `2` | Minimum packages for shotgun-surgery findings. |
| `shotgun_skip_conventional_names` | boolean | `true` | Skip conventionally named symbols in shotgun-surgery detection. |
| `cyclic_min_edges_per_direction` | number | `2` | Minimum edges per direction for cyclic-dependency findings. |
| `cyclic_skip_test_paths` | boolean | `true` | Skip test paths in cyclic-dependency detection. |
| `project` | string | — | Linked-project label from `.ministr.toml` `[[linked]]`. Omit for the primary corpus. |

## Returns

A `SolidResponse` with `findings` and `total`. Findings are labelled by
principle.

## Notes

- Detection is deterministic: thresholded graph and cosine-similarity signals,
  no LLM judgement.
- When a finding has more representatives than `representative_count`, the
  overflow is reported via `*_omitted` fields.
- Responses are wrapped in `ToolResponse<T>`, carrying `coherence_alerts`,
  `indexing_in_progress`, and `next_actions`.
- Soft errors (`ok: false`) never cancel sibling parallel calls.
