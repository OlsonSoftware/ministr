---
title: ministr_run_status
description: Poll a run's status, exit code, duration, and captured bytes.
---

Polls the status of a run started with `ministr_run`. Reports whether the run
is running, exited, killed, or timed out, along with its exit code, duration,
and total captured bytes.

## Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `run_id` | string | — | The run to poll. |

## Returns

A `RunStatusResponse` with:

- `run_id`
- `status` — one of `running`, `exited`, `killed`, `timed_out`
- `exit_code` (optional)
- `duration_ms` (optional)
- `bytes_total`

## Notes

- Useful for polling background runs started with `ministr_run`'s
  `background: true`.
