---
title: ministr_bridge
description: Query cross-language binding links between exported symbols and their consumers across language boundaries.
---

## Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `query` | string | no | Filter by binding key or symbol name (case-insensitive substring match) |
| `bridge_kind` | string | no | Filter by kind: `tauri_command`, `tauri_event`, `napi`, `wasm_bindgen`, `pyo3`, `http_route`, `ffi` |
| `language` | string | no | Filter to links involving this language (e.g. `rust`, `typescript`, `javascript`, `python`) |
| `file_path` | string | no | Filter to links where either endpoint is in this file path |
| `project` | string | no | Linked-project label. Omit for the session's primary corpus. |

## Response

Wrapped in the [standard response envelope](/docs/tools#response-envelope). The `result` payload:

```json
{
  "coherence_alerts": [ /* only when content changed; see envelope */ ],
  "result": {
    "links": [
      {
        "kind": "tauri_command",
        "confidence": 1.0,
        "export": {
          "binding_key": "open_file",
          "symbol_name": "open_file",
          "file": "src-tauri/src/commands.rs",
          "line": 42,
          "language": "rust"
        },
        "import": {
          "binding_key": "open_file",
          "symbol_name": "invoke",
          "file": "src/lib/api.ts",
          "line": 15,
          "language": "typescript"
        }
      }
    ],
    "total": 1
  }
}
```

### `result` fields

| Field | Description |
|---|---|
| `links[].kind` | Binding mechanism (`tauri_command`, `tauri_event`, `napi`, `wasm_bindgen`, `pyo3`, `http_route`, `ffi`) |
| `links[].confidence` | Combined link confidence (1.0 = exact match; lower values are case-normalized or semantic fallbacks) |
| `links[].export` | Producing side — see endpoint fields below |
| `links[].import` | Consuming side — same endpoint shape |
| `total` | Total matched links returned |

#### Endpoint fields (`export` and `import`)

| Field | Description |
|---|---|
| `binding_key` | The shared key that pairs export with import (e.g. the Tauri command name) |
| `symbol_name` | Local symbol name on this side (e.g. the Rust function for the export, the JS `invoke` callsite for the import) |
| `file` | Source file containing this endpoint |
| `line` | 1-based line number of the endpoint in `file` |
| `language` | Source language (e.g. `rust`, `typescript`, `javascript`, `python`) |

### Supported Frameworks

| Framework | Export Side | Import Side |
|---|---|---|
| Tauri | `#[tauri::command]` | `invoke('name', ...)` |
| napi-rs | `#[napi]` | `import { name }` from native module |
| PyO3 | `#[pyfunction]`, `#[pymethods]` | `from module import name` |
| wasm-bindgen | `#[wasm_bindgen]` | JS/TS import from WASM module |
| HTTP routes | `#[get]`, `#[post]`, etc. | `fetch('/path')`, `axios.get('/path')` |

## Behavior

- Links are resolved with exact match first, then case-normalized, then semantic embedding fallback
- Confidence scores below `0.85` indicate fuzzy matches that should be verified
- Use `ministr_references` with a symbol ID to find references without the cross-language filter
