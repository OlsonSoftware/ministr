> List the current project plus any linked projects you can query in this session. Pass a returned `label` as the `project` argument to other tools; omit `project` for the current one. Linked projects come from `.ministr.toml` `[[linked]]` entries (or `ministr_clone` results).

None.

## Returns

A `ProjectsResponse` with `projects` and a `hint`. Each project entry carries an optional `label` and an `is_current` flag. The primary corpus appears as `label: null`, `is_current: true`.

## Notes

- Linked projects come from `.ministr.toml` `[[linked]]` entries, or from `ministr_clone` results.
- Use this at the start of a session to discover which `project` labels are valid for query tools such as `ministr_survey`, `ministr_read`, and `ministr_toc`.
