ministr is a code intelligence MCP server. It gives AI coding agents AST-level understanding of your codebase: semantic search across code and docs, symbol-level navigation, real reference graphs, and cross-language bridge detection across **13 binding kinds** (Tauri, napi-rs, PyO3, wasm-bindgen, gRPC, HTTP routes, FFI, and more) — across 40+ languages, all local.

## What ministr gives your agent

- **Symbol navigation** — find and trace functions, types, and callers via `ministr_symbols` / `ministr_references` / `ministr_definition`
- **Cross-language bridge detection** — follows a call across Tauri / napi-rs / PyO3 / wasm-bindgen / gRPC / HTTP / FFI boundaries via `ministr_bridge`
- **Hybrid semantic search** across code and docs via `ministr_survey` — meaning and keyword together, ranked
- **Multi-resolution results** — summary, section, claim, symbol, or full source — the slice that matters, not a file dump
